8. Variables

Variables allow data to be stored within a program.

Python supports various data types the five standard types are

Numbers

String

List

Tuple

Dictionary

The type is used to define the storage method and operations that can be carried out on the variable.

Immutability

Every object is defined as being immutable this means that it is unchangeable. For the core types lists and dictionaries can be changed while numeric values, strings and tuples are not changeable. In most case this isn't really noticeable in you code as you can assign a new value to a variable with the same name.

Numeric Variables

Python supports the numeric variable types int, float and complex.

Numeric variables are created when a value is assigned to them.

var1 = 20

The example above will assign the value 20 to the variable var1.

If you want to delete a variable then this can be done with the del statement, so to delete the variable var1 defined above you would use

del var1

Multiple variables can be deleted by separating them with commas.

del var1, var2

int (signed integers)

Integers or ints are positive or negative whole numbers and so are defined without a decimal point. Unlike other programming languages in Python there is no limit to the value that an integer can represent so they have unlimited size.

#create integer variables

intVar1 = 10

intVar2 = -786

intVar3 = 0xff

float (floating point real values)

Often called floats these are floating point or real numbers and so have a real integer and fractional part separated by a decimal point. Floats can also be defined with scientific notation using e or E.

#create float variables

floatVar1 = 0.0

floatVar2 = -89.5

floatVar3 = 89.4-e45

floatVar3 = 56.2e10

Complex (complex numbers)

Complex numbers have a real and imaginary part and are defined in the form a + bj where a is the real part and b the imaginary part. The imaginary part j represents the square root of -1.

#create complex variables

complexVar1 = 78.67j

complexVar2 = 78.23 + 0j

complexVar3 = 34.2e-3 + 23.6e3j

complexVar4 = 3.14J

Strings

Strings are defined by assigning a set of characters within quotation marks.

myString = 'Python Programming'

If you want to extract part of the string this can be done using the slice operator the first character in a string is at index 0, the following shows you how to get the character at index [1].

myString = 'Python Programming'

myNewString = myString[1]

The variable myNewString will now be set to 'y'.

To get a set of characters then the syntax of the slice operator is [start index : end Index]

myNewString = myString[8 : 15]

myNewString = myString[8 : ]

In the second case where no value is specified after the colon then myNewString will be the contain everything from character 8 to the end of the string. Minus numbers can also be used in the splice operator these work backwards from the end of the string rather than the beginning.

myNewString = myString[-8 : -3]

myNewString = myString[-1]

Strings can be concatenated together using the + operator.

myNewString = myString + 'is cool'

Strings also have a repetition operator *.

myNewString = myString * 2

The following example code shows strings in action

# File: stringExample.py

 

myString = 'Essential Python 3'

# Copy the 2nd character only

myNewString = myString[1]

# Print the character

print(myNewString)

# Print 'Program'

print(myString[7: 14])

# Print 'Programming'

print(myString[7:])

# Print 'min'

print(myString[-4: -1])

# Create 'Python Programming is cool'

myNewString = myString + 'is cool?'

# Print 'Python Programming is cool'

print(myNewString)

# Print two copies of the string

print(myString * 2)

# Print the length of the string

print(len(myString))

The contents of a string cannot be changed once it has been defined so the following would create an error.

myNewString[2] = “y”

Strings in Python are immutable (unchangeable) so when an operation is performed on a string its result is a new string.

The [ : ] operators described above are all sequence operators which will also work on other sequences such as tuples and lists which are described later on in this book.

String Methods

Strings also have additional methods which can be used to find, replace and do other operations .

The length of a string can be found with the len method.

length = len('Test String')

The type of string can be determined using the following methods isalpha, isdecimal, isdigit, isidentifier, islower, isupper, isnumeric, isprintable, isspace and istitle.

# File: stringMethods1.py

 

myString = "PythonProgramming"

myNumberString = '1234'

 

if myString.isalpha():

print("myString is alpha")

else:

print("myString is not alpha")

 

if myNumberString.isnumeric():

print("myNumberString is numeric")

else:

print("myNumberString is not numeric")

To find a string within another string the find method can be used the following code will find the sub string “Prog” within the string variable myString.

myString.find("Prog")

The replace method can be used to replace part of a string with another string this method requires two arguments the sub string to find and the string to replace it with.

myString.replace("Programming", "Coding")

To split up a string the split method is used, this results in a list were each item is a sub string of the initial string. If split is called without a argument then it will split the string up at every space alternatively a character can be defined which will be used as a separator to split the string.

myList = myData.split(",")

The following code shows some of the methods described above in action.

# File: stringMethods2.py

 

myString = 'Python Programming'

myData = 'Jon,Simon,Jane,Mark'

 

# find the position of Prog

print('Prog is at index', myString.find('Prog'))

 

# replace a sub-string

replaceString = myString.replace('Programming', 'Coding')

print(replaceString)

 

# Make string upper case

upperString = myString.upper()

print(upperString)

 

# capitalise string

capString = upperString.capitalize()

print(capString)

 

# Split the string into a list

myList = myData.split(',')

for item in myList:

print(item)

 

# split using default space

myList = myString.split()

for item in myList:

print(item)

String Formatting

The string format method allows you to create strings from a template and data.

# File: stringFormattingMethods1.py

 

# define the templates

stringTemplate1 = "The {0} is {1}"

stringTemplate2 = "{action} like an {type}"

 

# print formatted data

print(stringTemplate1.format('Price', 2.50))

print(stringTemplate2.format(action="Walk", type="Egyptian"))

In the code above the first template uses indexes within curly braces these are replaced with the data in the call to the format method so stringTemplate1 will be ' The price is 2.50'. In the second template keywords are used to pass the data into the string.

The format method supports specific formatting for different data types this allows for alignment, padding and formatting of numeric values. The formatting information is placed within the curly brackets after the field name and is separated from the field name by a colon.

stringTemplate1 = "The {0:20} is {1:10}"

In the above example the template will pad field 0 up to twenty characters and field 1 up to ten characters. The syntax of the format method is

[[fill]align][sign][#][0][width][.precision][typecode]

The align flag can be < left alignment, > right alignment, ^ centre alignment and = padding after a sign character, a padding character can be added before the align flag.

stringTemplate1 = "{0:<10} like an {1:>15}"

stringTemplate1 = "{0:.<10} like an {1:>15}"

So the first line of the code above would align the first item to the left and pad it with 10 spaces and the second item would be padded with 15 spaces and aligned to the right. The second line would do the same but the first item would be padded to the left with '.'.

Numbers can be formatted within the format template, to format a floating point number to be displayed with three decimal places the following code would be used.

stringTemplat3 = 'Floating Point {0:.3f}'

Floating point numbers can also be formatted using e for exponential formatting and g for default floating point.

Integers can be formatted by using b for binary, X hex or o for octal values.

# File: stringFormattingMethods2.py

 

# template pad the first field to the left with 10 spaces

# and pad the second field to the right with 15 spaces

stringTemplate1 = "{0:<10} like an {1:>15}"

 

# template pad the first field to the left with 10 exclamation marks

# and center the second field with 15 question marks

stringTemplate2 = "{0:!<10} like an {1:?^15}"

 

# template display 3 floats first one to 3 decimal places

# second one as standard float and the third in exponential

# format aligned right padded with zeros

stringTemplate3 = "Floats {0:.3f} {1:g} {2:0>20e}"

 

# template display 3 integers first one in octal padded to the left with 5 spaces

# second one as hex value the third in binary

stringTemplate4 = "Integer {0:<5o} {1:X} {2:b}"

 

print(stringTemplate1.format("Walk", "Egyptian"))

print(stringTemplate2.format("Walk", "Egyptian"))

print(stringTemplate3.format(2.5687, 2.5687, 2.5687))

print(stringTemplate4.format(125, 125, 125))

Variable Scope and Namespaces

Variables are identifiers that map to objects. A name space is a dictionary of variable names and their values.

Python can access variables that are in the global and local namespace. If there is a global and local variable in scope with the same name then Python will use the local variable. Every function and class method has its own name space. Python assumes that any variable within a function is local to that function. If you want to use global variables in a function then you need to use the global statement.

In the following code total is defined as being global in the function addtototal, if it was not defined as global then the interpreter would give an error as total would not have been defined before it was used.

total = 0

# tell python that total is the global variable

global total

total = total + a

 

for i in range(0, 10):

addtototal(i)

print(total)

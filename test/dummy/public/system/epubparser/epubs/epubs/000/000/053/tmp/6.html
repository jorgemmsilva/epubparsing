5. Python Basics

Using the Interpreter Interactively

Start up the interpreter and type

print ('Hello World')

followed by 'enter', the interpreter should display a new line

Hello World

The interpretor has interpreted the print command and has printed 'Hello World' to the screen.

To exit the interpretor use type exit() or quit();

Using the Interpreter with a Script

While the interpreter can be used interactively this is not very useful if every time you want to run a program you need to type all the commands in one line at a time. For this reason writing Python code in script files makes a lot more sense.

To write a simple script use a text editor (e.g. Notepad++) to enter the following

print ('Hello World')

and save the file as HelloWorld.py all Python script files have the extension .py.

Open a command window and navigate to the directory with the file in it and type

python HelloWorld.py

The Python file should run and you should see the following displayed on the screen.

Hello World

Alternative to navigating to the directory with the file in you could include the files path, so if you code was in c:users\pythoncode\

python c:users\pythoncode\HelloWorld.py

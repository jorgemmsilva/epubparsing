12. Program Flow Control

This section covers flow control there are two methods of flow control one is a way to control the way a program is looped and the other type of control allows the program to make decisions. Python provides implementations of while and for loops which are similar to those found in other programming languages for creating loops and the if statement for decision making.

While Loop

A while loop repeats as long as an expression remains true. When the expression evaluates to false then the program skips the loop and the next statement after the loop is processed.

while expression:

suite

The example below shows how a while loop can be used to count from 0 to 10.

# File: whileExample.py

 

count = 0

while count <= 10:

print(count)

count += 1

print("Count Finished")

In the above example count is set to 0, the while statement checks that count <= 10 if it is then the block of code after the while statement is processed in the above example the current count value is printed and then it gets incremented. The program then jumps back to the while statement and count is evaluated again.

When count is no longer less than or equal to 10 then the code in the block is skipped and the code to print 'Count Finished' is run.

When constructing while loops you need to make sure that the expression will eventually evaluate to False, if it doesn't the the loop will never exit. If this is the case then you can exit the program using CTRL + C.

If there is only one line of code in the block then the while statement and the line of code can be on the same line.

For Loop

The for loop allows you to run a block of code multiple times while a condition remains True. In Python the operator iterates through the items of a sequence.

for iter in sequence:

suite

When the for loop is first run the value of the iterator will be given the first value in the sequence, when the statements in the suite have been run then the program will jump back to the for statement and the iterator variable will be given the next value in the sequence. If there are no values left in the sequence then the suite will be skipped and the next line after the suite will be run. The various ways a loop can be controlled are shown below.

# File: forExample.py

 

myList = ["Susan", "Dave", "Jane", "Mark"]

 

# Loop through a list

for loopIter in myList:

print(iter)

print("Loop1 Finished")

 

# Loop through a string

for character in "Test Loop":

print(character)

print("Loop2 Finished")

 

# loop through a range

for count in range(4, 7):

print(count)

print("Loop3 Finished")

 

# loop through a range defaulting from 0

for count in range(5):

print(count)

print("Loop4 Finished")

 

# loop through a range with a differnt step value

for count in range(0, 10, 2):

print(count)

print("Loop5 Finished")

In the first for loop the program iterates through all the values in the list. The second example shows the for loop iterating through the characters in a string. The last three loops use the range statement to control the loop. The first of these displays the values in the range between the values 4 and 7. In the second example the numbers 0 to 4 will be displayed and in the final example of using the range statement 0 to 8 in steps of 2 will be displayed.

Note the for loop terminates when it reaches the last value in the range statement, so in the following example the values 4, 5, 6 will be displayed.

for count in range(4, 7):

print (count)

The range statement is mainly used in for loops but it can be used if there is a requirement to produce a list of integers.

myList = list(range(5:10))

is the equivalent of

myList = (5, 6, 7, 8, 9)

Offsets and Items

The enumerate function can be used with the for loop to give the index of an object that is being iterated through.

# File: enumerate.py

 

myText = "Gronk"

 

for (index, value) in enumerate(myText):

print(index, value)

The above code would produce the following output.

0 G

1 r

2 o

3 n

4 k

Continue Statement

The continue statement is used within loops and causes the program to jump back to the start of the loop, it can be used in for or while loops.

# File: continueExample.py

 

myList = ["Jon", "Alison", "Dave", "Kate"]

 

# Loop through a list don't print Dave

for loopIter in myList:

if loopIter == "Dave":

continue

print(loopIter)

 

# Loop while i is less than ten ignore 3

i = 0

while i < 10:

i += 1

if i == 3:

continue

print(i)

Break Statement

The break statement terminates the current loop, none of the commands after the break statement in the loop will be called.

# File: breakExample.py

 

myList = ["Sarah", "Dave", "Jane", "Mark"]

 

# Loop through a list don't print Dave

for loopIter in myList:

if loopIter == "Dave":

break

print(loopIter)

print("Exit")

 

# Loop while i is less than ten ignore 3

i = 0

while i < 10:

i += 1

if i == 3:

break

print(i)

print("Exit")

The output from the above script is shown below note that in this case when break statement gets called the loops stop running.

Loop Statements and Else

The else statement can be used with both for and while loops. With the for statement the else statement gets executed when the loop has finished iteration. With the while statement the else statement is called when the while expression evaluates to false.

# File: loopElseExample.py

 

# Loop with a for statement

for count1 in range(0, 5):

print(count1)

else:

print("For End1")

 

# loop again exit with break

for count1 in range(0, 5):

if count1 == 2:

break

print(count1)

else:

print("For End2")

 

# Loop with a while statement

count2 = 0

while count2 < 4:

print(count2)

count2 += 1

else:

print("While End1")

 

# loop again exit with break

count2 = 0

while count2 < 4:

print(count2)

if count2 == 2:

break

count2 += 1

else:

print("While End2")

In the above output it can seen that the else statement only gets called when the number of iterations is completed and not simply when the loop is exited due to the break statement.

If Statement

The if statement allows your program to make decisions. The general format of an if statement is

if expression1:

suite1

elif expression2:

suite2

else:

suite3

When an if statement is run the suite associated with the first expression which evaluates to true will be run if none of the expressions evaluate to true then the suite in the else statement will run, the elif and else statements are optional. The following code contains two nested loops, if statements are used to print different messages dependant on the values of the two loops.

# File: ifExample.py

 

for j in range(5):

for i in range(10):

if i == 2:

print("Two")

elif i == 3 and j == 4:

print("Three and Four")

# Check if j is even

if j == 2 or j == 4:

print("Even")

else:

print("Odd")

if / else Ternary Expression

The ternary expression can be used in place of a simple if statement where the value of a variable needs to be set dependant on an expression for example using the if statement the code might be as follows.

number = 2

if number == 2:

result = 10

else:

result = 20

print (result)

The equivalent ternary expression would be.

number = 2

result = 10 if number == 2 else 20

print (result)

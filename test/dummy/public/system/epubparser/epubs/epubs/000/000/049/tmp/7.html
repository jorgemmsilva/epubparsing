6. Python Help

Python includes a help system which allows you get details of libraries and keywords. To find extra information on a function in the Python command window type help followed by the function name in parenthesis, for example to see information on the print function type.

help(print)

If you not a hundred percent sure what you are looking for then you can use the interactive help. To access this type help() which should bring up a simple menu based help system. To exit interactive mode type quit() or use Control-D on a Mac or UNIX or Ctrl-Z for windows.

To see a list of the methods and functions in a module from the command line then you first need to import the module and then you use the dir command. For example on the Python command line type

import commandparser

dir(commandpaser)

The various methods and functions for the configparser module should then be displayed on the screen. This is not limited to the inbuilt modules you can also use dir for your own modules or those written by anyone else.

9. Data Types

Lists

Lists are a compound data type that allow storage of different data types. The size of a list can be change after its been defined and the values stored within it can also be changed.

The values stored in a list can be accessed using the slice operator the first item in the list is at index zero.

Lists are defined by specifying the values within square brackets.

#create a list with some names and ages

mylist = [“Jon”, 22, “Dave”, 33]

 

#create an empty list

mylist2 = []

Similarly to strings lists support the concatenation and repetition operators. The following code demonstrates the use of lists.

# File: listExample.py

 

myList = ["Jon", 22, "Dave", 33]

anotherList = ["Simon", 33, "Mark", 45]

 

# display the mylist

print(myList)

 

# display the last 2 items in myList

print(myList[2:4])

 

# Create new list from 2 items from anotherList and myList

newList = myList[0:2] + anotherList[2:4]

 

# display the newList

print(newList)

 

# Change the first 2 items in myList

myList[0] = "Tan"

myList[1] = "Unknown"

 

# display the mylist

print(myList)

 

# iterate through the list

for x in myList:

print(x)

The list can be increased in size using the methods append, extend and insert. The append method adds a new item to the end of the list where as the extend method adds multiple items to the end of the list.

myList.append("Jane")

myList.extend(["28", "James", "12"])

The insert method is used to insert an object into the list at a particular index.

myList.insert(5, "Steve")

The list can have objects removed with the del, pop and remove methods. The del method is used to delete either a single or multiple items using the slice operator from the list where as pop will remove the last item on the list or an item at a particular index.

del myList[1:3]

myList.pop()

myList.pop(1)

The remove method removes items using the items value so the following will remove all the items with the value 'Mark'.

myList.remove("Mark")

If the number of items in the list is required then this is done in the same way as with strings using len. The count method returns the number of times a particular value is in the list.

listlen = len(mylist)

numberofelevens = myList.count("11")

To search and find an index holding a particular value the index method can be used.

dataindex = myList.index("Tan")

To sort a list or reverse the order of a list the sort and reverse methods can be used.

myList.sort()

myList.reverse()

The following example shows the methods described above in action.

# File: listExample2.py

 

myList = ["Jon", "22", "Dave", "11", "Janet", "11", "Mark", "45"]

 

# append an object to the end

myList.append("Izzy")

print(myList)

print(len(myList))

 

# extend with multiple objects

myList.extend(["28", "James", "12"])

print(myList)

print(len(myList))

 

# insert an object at position 0

myList.insert(0, "Susan")

print(myList)

 

# delete object at 0

del myList[1]

print(myList)

 

# delete objects at pos 1 and 2

del myList[1:3]

print(myList)

 

# pop object of list

myList.pop()

print(myList)

 

# remove Mark from list

myList.remove("Mark")

print(myList)

 

# count the number of elevens

print("Number of 11's")

print(myList.count("11"))

 

# find the index of Izzy

print("Get Index of Izzy")

print(myList.index("Izzy"))

 

# Add another Izzy

myList.append("Izzy")

print(myList.index("Izzy"))

 

# sort the list

myList.sort()

print(myList)

 

# reverse the list

myList.reverse()

print(myList)

Since lists can contain any object it is possible to have a list containing other lists. In the following script myList contains another list containing the value 22, 23 and 24.

# File: listExample3.py

 

myList = ["Jon", [22, 23, 24], "Dave", 11, "Sarah", "11", "Mark", "45"]

 

# Display the list

print(myList)

 

# change the value in the list in the list

myList[1][2] = 999

 

# change a list item

myList[5] = "Test"

print(myList)

In the above example an item within the second list is accessed this is done using two slice operators.

Tuple

Tuples are similar to lists the main difference is while the values in a list can be changed the values in a tuple are fixed when it is defined and cannot be changed. Tuples are defined using parenthesis as opposed to the brackets used to define a list.

Similarly to strings and lists tuples support the concatenation and repetition operators. The following code shows a tuple in action.

# File: tupleExample.py

 

myTuple = ("Jon", 22, "Jane", 33, "Simon", 33, "Toyah", 45)

# Print myTuple

print(myTuple)

 

# Print 3rd and 4th items form myTuple

print(myTuple[2:4])

 

# Create a new tuple

newTuple = ("Susan", 22, "Rob", 33) + myTuple[2:4]

 

# Print the new tuple

print(newTuple)

 

# Iterate through the tuple

for i in newTuple:

print(i)

Although the values in the Tuple can't be changed the values of the items within it can be change. So in the following example the values within the list in the following code held in the tuple can be changed.

# File: tupleExample2.py

 

myTuple = ("Jon", 22, [2, 3, 4], 33, "Alison", 33, "Mark", 45)

# Print myTuple

print(myTuple)

 

# change a item within the list

myTuple[2][1] = 666

 

# Print myTuple

print(myTuple)

Dictionaries

Dictionaries consist of records each containing a key and a value. The following example defines a dictionary with the records containing names as keys and ages as numeric values. The keys do not need to be strings they can be any immutable object. Dictionaries have no concept of the order of the data within them.

myDictionary = {“Jon” : 22, “Simon” : 30, “Tom” : 55}

The records can be changed using the slice operator so to change the value of the record with the key 'Jon'.

myDictionary['Jon'] = 23

The following code shows how dictionaries can be used

# File: dictionaryExample1.py

 

myDictionary = {"Jon": 22, "Dave": 33, "Sarah": 33, "Mark": 45}

# Print myDictonary

print(myDictionary)

 

# Print the keys of myDictonary

print(myDictionary.keys())

 

# Print the values of myDictonary

print(myDictionary.values())

 

# Create a new dictionary

newDictionary = {"Becky": 22, "Rob": 33}

 

# Print the new dictionary

print(newDictionary)

 

# Change value on a record

newDictionary["Becky"] = "Unknown"

 

# Add a new record

newDictionary["Paul"] = 55

print(newDictionary)

 

# remove item

newDictionary.pop("Paul")

 

# Print the keys

print(newDictionary.keys())

 

# Print the values

print(newDictionary.values())

 

# Print the items

print(newDictionary.items())

 

# Print the number of items

print(len(newDictionary))

 

# combine dictionaries

myDictionary.update(newDictionary)

print(myDictionary)

 

# delete an item

del myDictionary["Dave"]

print(myDictionary))

The example above shows how dictionaries can change size after they have been defined. The pop and delete methods remove items specified by their key.

newDictionary.pop("Paul")

del myDictionary["Dave"]

New items are added by adding a key and value

newDictionary["Paul"] = 55

or directories can be combined with the update method

myDictionary.update(newDictionary)

Since dictionaries are unordered they do not support sequence operations.

The following example shows how dictionaries can contain more complex objects than a simple variable, it shows a dictionary containing a tuple with "Dog", "Cat", "Mouse" and also another dictionary with the sounds the animals make.

# File: dictionaryExample2.py

 

myDictionary = {"Jon": 22, "Jane": ["Dog", "Cat", "Mouse"], "Simon": {1: "Bark", 2: "Pur", 3: "Squeak"}, "Mark": 45}

 

# Print myDictonary

print(myDictionary)

 

# Print value with the key "Jane"

print(myDictionary["Jane"])

 

# Print value with the key 1 from "Simon"

print(myDictionary["Simon"][1])

10. Data Type Conversion

If you need to convert from one data type to another then Python will probably have a built in function to do the conversion the most commonly used conversions are shown below.

int(x, [.base]) Converts x to an integer, the base can be specified if x is a string.

a = int("22") # a will equal 22

float(x) Converts x to a float.

b = float("23.4") # b will be 23.4

complex(real, img) Creates a complex number.

c = complex(4, 7) # c will be a complex value

str(x) Converts x to a string.

d = str(234) # d will be '234'

eval(x) Evaluates a string x and returns an object.

f = eval ("2 + 4") # f will equal 6

chr(x) Converts integer x to a character.

g = chr(63) # g will be equal to '?'

ord(x) Converts single character x to its integer value opposite of chr().

h = ord('?') # h will be equal to 63

hex(x) Converts integer x to a hexadecimal string.

i = hex(255) # i will be 'FF'

oct(x) Converts integer x to a octal string.

j = oct(255) # j will be '377'

The following code shows an example of the above conversions in action

# File: conversionExample.py

 

# Convert float to int

intVar1 = int(2.4)

 

# Convert hex string to int

intVar2 = int("abc33", 16)

 

# Converts string to a float.

floatVar = float("2.34")

 

# Creates a complex number.

complexVar = complex(2.3, 3.4)

 

# Converts object to a string.

strVar = str("456")

 

# Converts object to an expression string.

complexStr = repr(complexVar)

 

# Evaluates a string and returns an object.

evalVar = eval("2 + 3")

 

# Converts integer to a character.

chrVar = chr(65)

 

# Converts single character x to its integer value.

ordVar = ord('A')

 

# Converts integer x to a hexadecimal string.

hexStr = hex(255)

 

# Converts integer x to a octal string.

octStr = oct(255)

 

print(intVar1)

print(intVar2)

print(floatVar)

print(complexVar)

print(strVar)

print(complexStr)

print(evalVar)

print(chrVar)

print(ordVar)

print(hexStr)

print(octStr)
